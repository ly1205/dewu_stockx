## Introduction - 介绍

该项目主要完成对得物及StockX交易数据的爬取与分析，对本地的交易数据进行分析生成一些数据分析图。



## Configuration - 配置

pip install -r requirements.txt

根据自己cookie配置conf.ini



## Summary - 概要

执行python web_app.py

执行python DuSpider.py

执行python StockxSpider.py
